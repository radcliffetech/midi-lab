About MIDI Lab

A browser-based MIDI controller demonstrating real-time WebSocket communication and MIDI output to DAWs like Ableton Live.

It allows users to interact with MIDI devices directly from their web browser, creating a responsive and accessible music experience.

This project is a simple, real-time browser-based MIDI controller built with Go, WebSockets, and Bootstrap.

Prototype for live performance and audience interaction applications. Written by Jeffrey Radcliffe
